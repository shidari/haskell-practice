"H"は 'H':[]なのでString
'H'は 'H'なのでChar

2001に対して"2001"は['2','0','0','1']なので、一致しない

[1,2,3] ++ [3,2,1]
1:([2,3] ++ [3,2,1])
1:(2:([3] ++ [3,2,1]))
1:(2:(3:([] ++ [3,2,1])))
= 1:(2:(3:[3,2,1]))
= 1:(2:([3,3,2,1]))
= 1:([2,3,3,2,1])
= [1,2,3,3,2,1]
"Hello" ++ "World!"
= "HelloWorld"(上と同様)
[1,2,3] ++ [] = [1,2,3]
"Hello" ++ "" ++ "World!"
= "Hello" ++ [] ++ "World!"
="HelloWorld!"
