例えばA = [a,b], B = [c,d]とすると

width A = (b - a) / 2, width B = (d - c) / 2 
また、A + B = [a + c, b + d]より
width (A + B) = ((b + d) - (a + c)) / 2 
= ((b - a) + (d - c)) / 2 
= width A + width B 

また A * B = [min(a * c, a * d, b * c, b * d), max(a * c, a * d, b * c, b * d)]なので当然
width A * width B /= width (A * B)
